# LayoutCore

This is the official **LayoutCore** plugin developed by plentymarkets. **LayoutCore** provides basic components to build template plugins for [plentymarkets 7](https://www.plentymarkets.eu/tour/) online stores.

## Requirements

This is a plugin for [plentymarkets 7](https://www.plentymarkets.com). **LayoutCore** is required to run the plentymarkets [Callisto 4](https://github.com/plentymarkets/callisto-4) plugin.

## Installing

For detailed information about plugin provisioning refer to [plentymarkets developers](https://developers.plentymarkets.com/dev-doc/basics#plugin-provisioning).
